#Telnet
#TelnetCommander
#RedirectOutput

Данная программа построена на работе с сокетами.

Класс Telnet - ядро программы. Данный класс содержит функции подключения к сессиям и отключения от них, функция отправки команды,
чтения и записи потокового ввода/вывода, и, наконец, функция парсинга telnet сессии, где главным условием является установленное соединение
с сервером.

Класс TelnetCommander (далее командер) - дословно - команды telnet. В нём содержатся команды для отправки на сервер, исходя из вендора оборудования.
Внутри командера создается объект класса Telnet для установки соединения. В параметры объекта передается IP адрес и порт для подключения (по умоолчанию 23).
